

PORTABLE VERSION (Only on WINDOWS)

'Z-Anatomy_Portable.7z' is a zipped file that contains both the file and a portable version of Blender.
This version is already configured and does not need to be installed.
It can be used directly, even from a USB Key.

	-Download and unzip the file 'Z-Anatomy_Portable.7z' (https://www.7-zip.org/download.html)

	-Open the 'Anatomy Blender' folder and double click on 'Z-Anatomy.exe'

	-Right-click on the icon that appears in the taskbar to pin it.

	-Right-click on the 'Z-Anatomy.exe' > Create shortcut. 
	 You can rename it 'Z-Anatomy' and drop it on your desktop
